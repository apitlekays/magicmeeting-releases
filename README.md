# MagicMeeting — Downloads

Public download host for **MagicMeeting**, a macOS meeting recorder with on-device
transcription, speaker diarization, minutes/action items, and research notes.

The application source lives in a separate private repository; this repo exists only
to distribute the built, signed, and notarized macOS installers.

## Download

Get the latest build from the [**Releases**](https://github.com/apitlekays/magicmeeting-releases/releases/latest) page.

Each release ships one **universal2** disk image (`MagicMeeting_<version>_universal.dmg`)
that runs natively on both Apple Silicon and Intel Macs.

## Install

1. Download the `.dmg`.
2. Open it and drag **MagicMeeting** to **Applications**.
3. Launch it.

The build is signed with a Developer ID certificate and notarized + stapled by Apple,
so it opens with no Gatekeeper prompt — online or offline.

## Requirements

- macOS 13 (Ventura) or later.
